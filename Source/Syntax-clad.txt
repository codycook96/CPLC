//Purpose: This document describes the syntax to be used for my C-PLCSim Commands

Comments:
//Comment
/*
    Block Comment
*/

+ Header 
    //Each .clad file shall begin wiht a header with the following tag elements:

    ++ List of External Inputs with Type and Size
        IN = { DI in1, DI[64] AI in2, SI in3 }

    ++ List of External Outputs
        OUT = { DO out1, AO out2, AO[4] out3 }

        //Possible Types are:
            DI - Digital Input (Boolian)
            DO - Digital Output (Boolian)
            AI - Analog Input (Floating Point)
            AO - Analog Output (Floating Point)

    ++ List of Variables with Type and Size
        VAR = { INT x, FLOAT y, STRING str, BOOL b, BOOL[4] b2 }

        //Possibly Types are:
            INT - Integer
            FLOAT - Floating Point
            STRING - character array
            BOOL - Boolian

+ Body
    Each .clad file shall have a body containing the ladder logic to be executed

    //Structural Elements
    ++ Start of new Rung
        + 
    
    ++ End of a Rung
        +

    ++ Wire Separator inside Rung (acts as white space, for asthetics)
        -

    ++ Wire Separator between Rung (acts as white space, for asthetics)
        |

    ++ Branch Down
        \

    ++ Branch Up
        /

        //Branches are paired up sequentially
            i.e the first branch down of Rung 3 will be paired with
            the first branch up of rung 4
        //A line may have branches to start rather than rung

    //Logical Elements
        e.g. [XYZ]<mytag><othervar>

    ++ Normally Open Contact
        []<name> 
    
    ++ Normally Closed Contact
        [/]<name>

    ++ Output Coil
        ()<name> 

    ++ Set or Latch
        (S)<name>

    ++ Reset or Unlacth
        (R)<name>

    ++ One-Shot Rising Edge
        [OSR]<name>

    ++ One-Shot Falling Edge
        [OSF]<name>

    //Incremental Elements
    ++ Timer On Delay (Turns on after "preset" time when rung high)
        [TON]<name><reset><time><preset>        //preset may be constant or INT variable

    ++ Timer Off Delay (Turns on immediately when rung high, stays on "preset" time after rung low)
        [TOF]<name><reset><time><preset>        //preset may be constant or INT variable

    ++ Retentative Timer (Rung out = rung in, runs time when rung high, only resets with reset)
        [RTO]<name><reset><time><preset>        //preset may be constant or INT variable

    ++ Count Up (Counts up when rung high)
        [CTU]<name><reset><count><preset>       //preset may be constant or INT variable

    ++ Count Down (Counts down when rung high)
        [CTD]<name><reset><count><preset>       //preset may be constant or INT variable

    //Comparison Elements
    ++ Equal, Not Equal, Greater Than, Greater Than Equal To, Less Than, Less Than Equal To
        [EQU]<name><var1><var2>
        [NEQ]<name><var1><var2>
        [GRT]<name><var1><var2>
        [GEQ]<name><var1><var2>
        [LES]<name><var1><var2>
        [LEQ]<name><var1><var2>

    //Arithmetic Elements
    ++ Add, Subtract, Multiply, Divide
        [ADD]<name><var1><var2><dest>       //dest = var1 + var2
        [SUB]<name><var1><var2><dest>       //dest = var1 - var2
        [MUL]<name><var1><var2><dest>       //dest = var1 * var2
        [DIV]<name><var1><var2><dest>       //dest = var1 / var2

    


        //Where name describes a digital tag that can be referenced elsewhere,
        //If tag is not defined as an input or output is defined internally
        //Only Contacts may be assigned external inputs
